# Task Manager App

A Task Management System built with **React**, **Node.js**, **Express**, and **MongoDB**.

## Features

- Create, Read, Update, Delete (CRUD) for Tasks
- Task status management: Pending, In Progress, Completed
- Task due dates
- API routes
- Responsive UI

---

## Tech Stack

### Frontend
- React
- Axios
- React Hooks
- CSS 

### Backend
- Node.js
- Express
- MongoDB (Mongoose)
- express-validator for input validation
